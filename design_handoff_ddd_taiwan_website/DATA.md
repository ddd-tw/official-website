# DDD Taiwan — 真實資料彙整（供實作填充）

## 社群簡介（官方文案來源：KKTIX 社群頁）
Domain Driven Design Taiwan Community 是基於對領域驅動設計的熱愛組成的交流社群，關注從業務角度出發、探索真正的商業需求與要解決的問題，透過跨組織協作與利益關係人交流，設計出滿足需求並創造價值的解決方案。「一起探索從 Problem Domain 去催生出 Solution Domain」。非營利、志工經營（無法開立發票）。成立於 2018 年。

- 2025 第五屆：「AI 時代軟體開發方法（AI-Era Software Development Practices）」，2025/11/8。主題含 AI 輔助開發、文件即程式碼、Incident Response 自動化。
- 2021：「足跡 Footprints」。海外講者：Kenny Baas-Schwegler（Event Storming 視覺化共創建模）、Dr. Carola Lilienthal（legacy system）、Adam Dymitruk（Event Modeling 創始人）。議程軌：Heuristics 啟發／Footprints 足跡。工作坊：千錘百鍊 Event Storming（全日六小時）、Domain Storytelling。
